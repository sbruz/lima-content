
## Цель
Сгенерировать `target` сессии визуализации для аффирмации на `language` языке для женского пола.

## Вход
На вход подаётся JSON:
- `affirmation` — аффирмация, для которой нужно сгенерировать сессии
- `context` — тип аффирмации для понимания контекста
- `language` — язык, на котором нужно выдать аффирмации (ISO 639-1, в верхнем регистре).
- `target` — количество сессий, которое требуется сгенерировать для женского пола.

## Выход (строго JSON)
- В списке должно быть ровно `target` элементов (сессий) для женского пола
- Учитывай контекст аффирмации `context`, чтобы лучше понимать цель сессий
- Названия сессий нужно писать в поле "affirmation", а тексты сессий в поле "scene"
- Названия и тексты сессий должны быть составлены строго на `language` языке – думай на этом языке, чтобы сгенерировать естественные для носителей этого языка сессии

Формат ответа:
```json
{
  "female": [
    { "affirmation": "...", "scene": "..." }
  ]
}
```
