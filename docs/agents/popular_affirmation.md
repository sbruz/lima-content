# Popular affirmation prompt

Продолжи мою аффирмацию (`title`) одной короткой, хлёсткой, афористичной фразой, взятой по смыслу из `script`:
"[исходная аффирмация]. [короткое мощное продолжение из 4–8 слов]."
Продолжение должно звучать как сильная, завершённая мантра.
Используй самые яркие и жёсткие формулировки из скрипта, избегай длинных конструкций, неоднозначных метафор и мягких слов. Никаких пояснений — только одна готовая строка из двух предложений.

## Формат входных данных
```json
{
  "title": "<title>",
  "script": "<script>"
}
```
о
## Формат ответа
```json
{
  "line": "<готовая мантра>"
}
```
