1) ### Шаг 4. Планирование эпизодов

Флаг generate_affirmations.
Логика:
- выбрать категории и подкатегории по диапазонам в конфиге
- выбрать тренеров из coach по списку в конфиге
- обратиться к LLM
   - поочереди для каждой категории, подкатегории и тренера передать категорию, подкатегорию, coaches.prompt с промтом описывающим роль, affirmations_base.md с промтом, описывающим формат входных/выходных данных
   - получить от LLM список из affirmations_per_subcategory аффирмаций для каждого пола (женщины, мужчины), соответствующих стилю тренера (coaches.prompt). Каждая аффирмация для женщин и мужчин - JSON из двух элементов: аффирмация и сцена на английском языке.

2) ### Шаг 5. Скрипты аффирмаций
Флаг script_affirmations.
Логика:
- выбрать категории и подкатегории по диапазонам в конфиге
- выбрать тренеров из coach по списку в конфиге
- выбрать список языков
- Обратиться к LLM
   - для каждого языка выполнить две последовательные подзадачи:
     1. **Перевод** (промпт `affirmations_script_translate.md`): подаём `affirmations.affirmation`, `coaches.prompt`, `target_language`. Для EN шаг перевода пропускаем.
     2. **Расстановка пауз** (промпт `affirmations_script_pauses.md`): на вход отдаём результат перевода (или исходник для EN), чтобы LLM расставила `\n`.
   - обе подзадачи возвращают одинаковый JSON `{"affirmation":{"female":{"title","script"},"male":{"title","script"}}}`
   - после шага пауз сохранить результат в поле `affirmations.script` (затереть, если непустое)
