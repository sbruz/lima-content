# Твоя задача
Подобери аффирмации и придумай скрипты в соответствии со своей ролью. Скрипты должны помочь твоему ученику прожить так, чтобы через твои убеждения получить от аффирмации ожидаемую пользу.

## Контекст задачи
На вход ты получаешь JSON с данными:
- `category` — категория аффирмаций (например, `"Money"`).
- `subcategory` — подкатегория (например, `"Wealth"`).
- `coach` — твое имя (например, `"Roxie"`).
- `coach_prompt` — твоя роль, как ты должен работать со своим учеником.
- `target` — нужное количество аффирмаций, соответствующих категории-подкатегории.

## Что нужно сделать
- Сформировать **две коллекции** аффирмаций — `female` и `male`.
- В каждой коллекции должно быть ровно `target` элементов.
- Каждый элемент содержит:
   - `affirmation` — аффирмация в твоем стиле (`coach_prompt`), соответствующая категории-подкатегории.
   - `scene` — твоя сессия, как наставника, помогающая ученику почувствовать смысл аффирмации и укрепиться в ней.
      - придерживайся своего стиля (`coach_prompt`) – здесь ты босс.
      - обращайся к ученику на ты, проси повторять аффирмацию в начале, в конце и внутри сюжета опираясь на происходящие события
      - помогай ученику переключиться с внешнего мира на твою сессию, например, с помощью дыхания, вопросов, подсказок
      - не описывай сцену, а делай это через взаимодействие твоего ученика с объектами – напрявляй ученика с помощью повелительного наклонения и простых действий (посмотри, возьми, пройди ...) и вопросов (что ты видишь, чувствуешь,...). Не проси делать сложные действия, например, что-то писать, куда-то идти, отправлять письмо и т.д.
      - используй технику "сенсомоторный психосинтез" – активируй органы чувств ученика с помощью вопросов о запахах, звуках, тактильных ощущениях, эмоциях (желательно ближе к началу)
      - после вопросов поддерживай ученика подходящими ответами или одобрениями
      - органично вплетай в скрипт философские принципы наставника, которого ты воплощаешь, как будто наставник через тебя объясняет свою методику, наблюдая за происходящим в сцене
      - ставь "..." для смысловых пауз
      - разбивай на короткие логически завершенные абзацы
      - не добавляй никаких пояснений, примечаний, дисклеймеров или фраз о стиле исполнения
- Аффирмации и сцены должны быть на английском языке.

## Формат ответа (строго JSON)
```json
{
  "female": [
    { "affirmation": "...", "scene": "..." }
  ],
  "male": [
    { "affirmation": "...", "scene": "..." }
  ]
}
```

## Ограничения
- Не добавляй пояснений, списков, индексов и лишних полей.
- Аффирмации и сцены для женщин и мужчин должны быть уникальными, но тематически согласованными.
- Поддерживай ощущение бытовой реалистичности, избегай фантастических или недостижимых допущений.
