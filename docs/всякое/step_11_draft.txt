Шаг 11: Написание вирусных аффирмаций

Цель: для каждого текста аффирмации в таблице affirmations_new_script с учетом пола и языка подобрать уникальную популярную аффирмацию.

Последовательность выполнения:
1) Запускаем цикл по Категориям/Подкатегории/Наставнику/Позиции с учетом фильтров в yaml.
Используем многопоточность.
2) Получаем значение поля affirmations_new.script
3) Запускаем цикл по языкам  с учетом фильтра в yaml
4) Репортим в лог, что мы сейчас обрабатываем в короткой форме
5) Для female/male отправляем в LLM по очереди title и script, плюс новый файл с промтом popular_affirmation.md.

Содержание директивы для LLM:
Продолжи мою аффирмацию одной короткой, хлёсткой, афористичной фразой, взятой по смыслу из скрипта: [исходная аффирмация] плюс [короткое мощное продолжение из 4–8 слов].
Продолжение должно звучать как сильная, завершённая мантра.
Используй самые яркие и жёсткие формулировки из скрипта, избегай длинных конструкций, неоднозначных метафор и мягких слов. Никаких пояснений — только одна готовая строка.

6) Сохраняем в json в поле popular_aff с группировкой по полу для обработанных языков. Если в этом поле ранее уже были данные, то обработанные языки затираем, а данные на других языках оставляем.

Продолжи мою аффирмацию дополнительным утверждением из текста скрипта. Используй любые фразы или идеи из скрипта, которые логично усиливают исходную аффирмацию. Продолжение должно быть лаконичным, ярким, контрастным, афористичным. Не осторожничай и не ограничивайся мягкими формулировками — используй самые сильные фразы из скрипта, если они подходят по смыслу. Например, для аффирмации "Я – магнит для денег" это может быть "Я – магнит для денег. Они приходят ко мне легко". Новая аффирмация должна звучать, как яркое расширение исходной аффирмации – лаконично, но более мощно, убедительно и трансформирующе.
